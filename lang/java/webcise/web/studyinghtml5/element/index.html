b, i, hr, s, small, u

これらの要素はHTML5で意味合いが変更され再定義された。

  • iは区別される要素。大抵italicで記述されます。
  • bは重要性は小さいが文体は変えたい要素。大抵太字になります。
  • sは正確でなくなった要素。大抵打ち消し線が表示されます。
  • uは伝わりにくい要素。下線が引かれます。
  • hrは意味的な気切りを示します。
smallはコピーライトなど注釈的な要素に対して使います。

audio

音楽を再生します。

最近のFirefoxは.mp3に対応しているので .mp3フォーマットのオーディオファイルを用意すれば主要ブラウザでは 同じ音楽を再生できるはずである。Safariは7.0.5でも.oggには対応していない。 preload="metadata"ではページ読み込み時と 音楽再生開始時の2回オーディオファイルのダウンロードが発生する。 preload="none"ではページ読み込み時にダウンロードが発生しないが 再生時間が分からない。
