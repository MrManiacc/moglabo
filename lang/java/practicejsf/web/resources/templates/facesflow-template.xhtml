デフォルトタイトル

faces-config.xmlのナビゲーションとh:commandButtonのactionが衝突すると
faces-config.xmlのナビゲーションが優先される。

以下の2つのリンクはactionの値をfaces-config.xmlのnavigation-rule要素に
設定を追加してページ遷移可能にしている。

↑POSTで遷移するからかURLが変更されない。その後相対パスのリンクをクリックすると404になってしまう。

a要素による通常のリンクは何の変わりも無く動作する。
