引数を渡してメソッド呼び出し

メソッドに引数を渡して呼び出せるようになったのはEL2.2から。

英語であいさつ:#{testbean2.greeting(false)}

日本語であいさつ:#{testbean2.greeting(true)}

デフォルトの言語であいさつ:#{testbean2.greeting()}

小さな乱数:#{testbean2.randomNumber(5)}

大きな乱数:#{testbean2.randomNumber(500)}

ランダムあいさつ:#{testbean2.greeting(testbean2.randomNumber(200) % 4 le 1)}
