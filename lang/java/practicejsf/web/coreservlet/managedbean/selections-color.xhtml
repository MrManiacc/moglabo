マネージドBean(select)

色選択

h:selectOneMenu

選択要素のvalueの型:String[]

h:selectOneMenuは何も選択されなかった時に先頭の要素が選択される。

f:selectItemsで選択された要素のvalueが親要素のh:selectXXXのvalueに設定される。

h:selectOneListbox

選択要素のvalueの型:List<String>

h:selectOneRadio

選択要素のvalueの型:Map<String, String>

h:selectOneListbox

選択要素のvalueの型:List<Color>

オブジェクトのListをvalueに指定した場合はラベルと値に使う文字列を指定する。
BeanのフィールドをString型のコレクションで保持しなくてもよい。
