このページはサービス提供者が提供するテンプレートを利用するクライアントのページになる。
ui:compositionの外側にあるのでこの文章は表示されない。
ui:compositionは先勝ちである。後に書かれたui:compositionは無視される。
異なるテンプレートを参照していても，より後ろのui:compositionは無視されてしまう。

このページのheadでCSSを読み込もうとしても無視される。テンプレートを介して
読み込まれるようにui:defineを使わなければならない。
基本的にテンプレートが本体でありクライアントのページはページ間の差異を表現するための
断片と考えた方がいい。
