ui:compositionの外側に存在する要素はFaceletsページに出力されない！
ui:defineで囲んでもダメである。

テンプレートにコンテンツを挿入

ui:defineにclass属性を指定してもスタイルが適用されない。

挿入されたサンプルタイトル ui:composition内に記述された要素はui:define内に含まれていなければ
Faceletsページに出力されない。つまりこの文章は表示されない。

挿入されたサンプルコンテンツ

ui:defineの内部のコンテンツを引数のようにテンプレートファイルへ渡せる。

ui:compositionを使ったFaceletsページの要素が表示されるには以下の条件を全て満たす必要がある。

  • ui:compositionの内部に記述されていること
  • ui:defineの内部に記述されていること
    •

    なおui:define内の要素に指定したclass属性に対するスタイルは適用される。
