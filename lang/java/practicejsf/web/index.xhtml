動作確認環境

JDKバージョン:8

アプリケーションサーバ:Tomcat8.0

JSFバージョン:2.2(JavaEE7サーバでの開発を想定)

ライブラリ

マネージドBeanを用いるJSF2.xを利用するだけならJSFのライブラリにのみパスを通します。
ただしJSTLも合わせて使いたい場合はjstl.jarとstandard.jarにもパスを通します。

  • JSFのライブラリ(必要なバージョン番号のライブラリをダウンロード)

マネージドBeanではなくCDI名前付きBeanを用いる場合はCDI関連ライブラリにパスを通す必要があります。

  • javax.inject(同名のzipファイルをダウンロード)
  • CDI API(「API」からダウンロード)
  • Weld(Flowスコープを使用しないなら不要？)
